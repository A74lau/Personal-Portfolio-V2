# AGENTS.md

## Project Context

This is a standalone React portfolio site (Vite + Tailwind CSS). 

Start with `README.md` for local setup and deployment.

## Key Files

- `src/`: frontend application source
- `src/pages/Home.jsx`: main portfolio page
- `src/components/`: portfolio sections and UI components
- `vite.config.js`: Vite configuration
- `public/`: static assets served as-is

## Working Notes

- Use `npm run dev` for local development
- Use `npm run build` to produce a static site in `dist/`
- Run `npm run lint` and `npm run typecheck` before finishing code changes
